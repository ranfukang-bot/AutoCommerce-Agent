# Agent Workflow

- product name
- category
- target market
- selling points
- price positioning

Output:

- user pain points
- localized strategy
- recommended video angles

## Viral Script Analysis Agent

Input:

- product insight
- product category
- target market

Output:

- 0-3s hook

- 8-15s conversion scene
- conversion triggers

## Video Prompt Agent

- preferred model

- VEO prompt

- risk profile

## Quality Review Agent

Input:

- generated prompt plan
- product category

Output:

- quality scorecard
- pass threshold
- revision rules

## Report Agent

Input:

- all previous outputs

Output:

- project summary
- model recommendation
- workflow plan
- cost control suggestions
- team management notes
