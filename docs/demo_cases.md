# Demo Cases

## Case 1: Oversized Graphic T-Shirt

Goal:

- Show fit
- Show print
- Show streetwear style

Best approach:

- VEO or Seedance
- Model wearing product
- Light turn, outfit detail, close-up print shot

Avoid:

- Large body movement
- Complex hand gestures
- Overly fast camera movement

## Case 2: Motorcycle Engine Oil

Goal:

- Build trust
- Show product bottle clearly
- Connect with motorcycle maintenance

Best approach:

- Product bottle hero shot
- Motorcycle background
- Performance-oriented visual mood

Avoid:

- Showing oil being poured into engine with complex hand motion
- False technical claims

## Case 3: Kitchen Storage Rack

Goal:

- Show before-after organization
- Show space-saving effect

Best approach:

- Messy kitchen opening
- Clean organized result
- Simple placement action

Avoid:

- Complex installation animation
- Fast hand assembly
