# Prompt Templates

## VEO Template

```text
Vertical 9:16 TikTok commercial video.
Product: {{product_name}}.
Category: {{category}}.
Target market: {{target_market}}.
Scene: realistic lifestyle environment, clean background, natural lighting.
Action: light, controlled movement that clearly shows the product.
Visual focus: {{selling_points}}.
Structure: 0-3s strong hook, 3-8s product value demonstration, 8-15s lifestyle close-up.
Camera: stable handheld feel, smooth push-in, clear close-up details.
Style: realistic, high-quality commercial, no distorted text, no extra logos, no watermark.
```
